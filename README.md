# Sticker IOS

An iOS app to add sticker packs to WhatsApp or WhatsApp Business.

## Tech Stacks
- Swift
- SwiftUI

## Features
- Support static and animated stickers
- Search sticker packs
- Add sticker packs to favorites
- Popular sticker packs
- Most liked characters
- All characters and character detail page

## How to use this repo

- Open the project in Xcode
- Put your sticker images in `Resource` folder
- Update the `sticker_packs.json` file
- Update the app icon and splash image
- Update the app name

## IMPORTANT !!!

This app is for learning purpose only! Do not submit it to App Store!
